作業効率を上げるために、便利なツールはどんどん活用しましょう。

基本的に無料のものをご紹介していますが、有料版でより使いやすいツールなどもありますので、自分が一番使いやすいものを探してみましょう。

データをダウンロードする際はセキュリティに留意し、著作権など確認しながら、各自の責任においてご利用ください。

コード関連

コーディング中に困ったことが出てきたらココ!まずは調べてみましょう。

  • MDN Web Docs

    【MDN Web Docs】

    世界中の開発者が公開しているWeb開発の学習プラットホーム。HTMLやCSSの解説など、Web開発に必要な知識が公開されています。

  • Qiita

    【Qiita】

    開発者が自身の知識や経験に関する記事を書き込むサイト。クラス名のリストや、バグの原因と解決法解説している記事など便利な情報がたくさんあります。

  • Can I use …

    【Can I use …】

    HTMLやCSSのブラウザ対応状況を調べることができます。

SNSプラグインツール

TwitterやFacebookなどを組み込むときに。

  • twitter

    【Twitterプラグイン生成ページ】

    Twitterのコード取得

  • Facebook

    【Facebookプラグイン生成ページ】

    Facebookのコード取得

You Tube、Facebook、Googleマップはそれぞれ表示させたいアカウントやページの「共有」ボタンからコードを生成できます。それぞれのやり方はGoogleで検索するとすぐに出てきますので調べてみてください。

その他便利ツール

画像変換や配色パターンなど、便利なツールを集めました。

  • I Love IMG

    【I Love IMG】

    画像の圧縮やJPGへの変換など、画像の加工ができます。

  • Font-family メーカー

    【Font-family メーカー】

    使いたいフォントを選んだらコードを生成してくれます。

  • ColorHunt

    【ColorHunt】

    4色の配色パターンが多く掲載されているサイトです。人気順や色のイメージから探すことができます。

  • Adobe Color

    【Adobe Color】

    画像から配色パターンやグラデーションを抜き出したり、1つの色から類似色などの配色パターンを作ってくれたり、かなり便利です。

  • Ultimate CSS Gradient Generator

    【Ultimate CSS Gradient Generator】

    グラデーション作成ツールです。感覚的に操作でき、コードを作成してくれます。
